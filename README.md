## Thinks

1. 服务端功能？
  目前项目定位在工具库，不考虑服务端功能．服务端能力由更上层的应用逻辑扩展．
2. 组件数据化管理？
  组件数据化管理会带来组件开发体验问题，需要引入更多的系统机制来保障，会增加项目复杂度。
  比如：本地开发组件后如何导入？本地组件如何调试？
